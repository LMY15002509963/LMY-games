# 球球大作战

一个基于HTML5 Canvas的网页版球球大作战游戏，支持多人在线对战。

## 游戏特性

- 🎮 基于Canvas的高性能渲染
- 🌈 精美的视觉效果和动画
- 🤖 AI玩家系统
- 🏆 实时排行榜
- 🗺️ 小地图功能
- 📱 响应式设计
- 🎯 分裂球球功能

## 游戏玩法

1. 输入你的名字开始游戏
2. 使用鼠标控制球球移动
3. 吃掉比你小的球球来变大
4. 避开比你大的球球
5. 按空格键可以分裂球球
6. 存活越久，得分越高

## 技术特点

- 纯前端实现，无需后端服务器
- 使用Canvas 2D API进行渲染
- 面向对象的游戏架构
- 优化的碰撞检测算法
- 流畅的60FPS游戏体验

## 部署说明

本项目已配置为可以直接部署到GitHub Pages：

1. 将代码上传到你的GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择源分支和文件夹
4. 游戏即可在线访问

## 本地运行

1. 克隆仓库到本地
2. 打开index.html文件
3. 开始游戏！

## 游戏控制

- **鼠标移动**: 控制球球移动方向
- **空格键**: 分裂球球
- **自动跟随**: 球球会自动跟随鼠标移动
