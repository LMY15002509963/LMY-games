# 球球大作战 - 极速竞技 🎮

一个功能丰富的多人在线球球大作战游戏，支持实时对战、智能AI玩家和流畅的游戏体验。

## 🌟 游戏特色

- **多人联机对战** - 支持最多50人同时在线游戏
- **智能AI系统** - 4种不同性格的AI玩家，具有高级狩猎和逃生策略
- **流畅操控** - 高灵敏度鼠标控制，无延迟响应
- **丰富视觉效果** - 粒子特效、动态阴影、脉动食物
- **实时排行榜** - 追踪玩家得分和排名
- **小地图系统** - 全局视野，掌握战场动态

## 🎮 游戏操作

- **鼠标移动** - 控制球球移动方向
- **空格键** - 分裂球球
- **W键** - 发射质量
- **ESC键** - 暂停/继续游戏

## 🚀 快速开始

### 本地运行

1. 安装依赖
```bash
npm install
```

2. 启动服务器
```bash
npm start
```

3. 打开浏览器访问
```
http://localhost:3000
```

### 开发模式
```bash
npm run dev
```

## 🛠️ 技术栈

- **前端**: HTML5 Canvas, JavaScript ES6+, CSS3
- **后端**: Node.js, Express.js
- **实时通信**: Socket.IO
- **跨域支持**: CORS

## 🎯 游戏策略

### AI玩家类型

1. **攻击型 (35%)** - 积极狩猎，擅长分裂攻击
2. **防守型 (35%)** - 谨慎移动，优先安全食物
3. **平衡型 (15%)** - 攻守兼备，灵活决策
4. **狩猎型 (15%)** - 专门追击小型玩家

### 生存技巧

- 小心比你大的玩家
- 利用分裂快速逃生
- 发射质量可以引诱敌人
- 观察小地图了解全局情况

## 🌐 在线体验

游戏支持自动替换AI玩家功能：
- 当有真人玩家加入时，会自动替换AI玩家
- 当真人玩家离开时，会补充AI玩家
- 确保游戏始终有足够的参与者
