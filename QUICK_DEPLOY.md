# 🚀 立即部署指南

## 方法1：使用Vercel (最快) ⭐

1. **访问Vercel官网**
   - 打开 https://vercel.com
   
2. **连接GitHub**
   - 使用GitHub账号登录
   - 授权访问您的仓库

3. **导入项目**
   - 选择 `LMY-games` 仓库
   - 确认项目设置
   - 点击 "Deploy"

4. **完成！**
   - 等待1-2分钟部署完成
   - 您的网站将自动上线

## 方法2：使用GitHub Pages

1. **进入仓库设置**
   - 访问 https://github.com/LMY15002509963/LMY-games
   - 点击 "Settings" 标签

2. **启用Pages**
   - 在左侧菜单找到 "Pages"
   - Source选择 "Deploy from a branch"
   - Branch选择 "main"，文件夹选择 "/(root)"
   - 点击 "Save"

3. **等待构建**
   - 几分钟后网站将在以下地址上线：
   ```
   https://lmy15002509963.github.io/LMY-games/
   ```
